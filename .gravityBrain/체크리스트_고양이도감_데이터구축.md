# CAT-AI-PEDIA 고양이 도감 데이터 구축 (50종)

## 📌 개요
- 목표: SEO 최적화 및 프리미엄 예술 도감 컨셉의 50종 고양이 JSON 데이터 구축
- 핵심 요구사항:
  1. 다국어 지원: 모든 텍스트 필드에 `{ "en": "...", "ko": "..." }` 구조 필수 유지.
  2. 로컬 렌더링 연동: 에이전트는 이미지를 직접 생성하지 않고, 로컬 RTX 5080 환경에서 사용할 수 있는 초고화질 영문 프롬프트를 JSON 내(혹은 별도)에 제공.
  3. 로컬 정보 포함: 한국 내 입양 인프라 및 아파트 적응 팁 등 한국 사용자 타겟 롱테일 키워드 포함.

## 📝 작업 단계 (MASC 검증 완료)
- [ ] **Step 1**: 다국어 JSON 스키마(Template) 확정 (기존 `{ en, ko }` 구조 + 신규 필드)
- [ ] **Step 2**: 50종 품종 리스트 확정 (이전 제시한 1~50위 기반)
- [ ] **Step 3**: 1~10위 품종 다국어 데이터 및 이미지 생성용 프롬프트 추출 (배치 작업)
- [ ] **Step 4**: 추출된 프롬프트를 바탕으로 사용자가 로컬(ComfyUI/Midjourney)에서 이미지 병렬 생성
- [ ] **Step 5**: 11~50위 품종에 대해 Step 3~4 반복
- [ ] **Step 6**: 최종 생성된 JSON 파일과 `Nyan/` 폴더 이미지의 무결성 테스트 (프론트엔드 연동)

## ⚠️ 주의 사항
- 에이전트는 `generate_image` 도구를 사용하지 않음.
- 기존에 덮어쓴 `ragdoll.json`과 `british-shorthair.json`은 새 다국어 스키마에 맞춰 재작성 필요.
