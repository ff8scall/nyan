# SEO Strategy

## 1. SEO Goal

Build an evergreen bilingual search presence around cat breed education, breed comparison, and lifestyle-based cat selection.

Primary objective:
- Capture long-tail informational search traffic in English and Korean.

Secondary objective:
- Convert search visitors into comparison users, quiz users, newsletter subscribers, and eventually affiliate clicks.

## 2. SEO Positioning

The site should compete as a practical breed decision guide, not just a visual list of cat breeds.

Core SEO angle:
- `cat breed + personality`
- `cat breed + care`
- `cat breed + apartment`
- `cat breed + family`
- `cat breed + shedding`
- `cat breed comparison`
- `고양이 품종 + 성격`
- `고양이 종류 + 추천`
- `초보자 고양이 품종`

## 3. Sitewide Technical SEO

### Required
- Unique title per page.
- Unique meta description per page.
- Canonical URL.
- Hreflang between English and Korean equivalents.
- XML sitemap.
- Robots.txt.
- Open Graph image.
- Fast image loading.
- Mobile-first layout.
- Semantic HTML headings.

### Hreflang
Each page should reference:
- English version.
- Korean version.
- Optional x-default.

Example:
- `/en/breeds/maine-coon`
- `/ko/breeds/maine-coon`

### Canonical
Canonical should point to the current language version, not always English.

## 7. Structured Data

### Recommended Schema Types
- BreadcrumbList.
- FAQPage where visible FAQs exist.
- Article or WebPage.
- ItemList for breed index.

### Caution
There is no perfect standard schema for cat breed pages. Avoid forcing inaccurate schema. Use general WebPage/Article plus FAQ and Breadcrumb where appropriate.

## 8. Internal Linking Strategy

### Breed Page Internal Links
Each breed page should link to:
- breed index,
- 3-5 similar breeds,
- comparison page with related breed,
- quiz page,
- relevant guide pages.

### Guide Internal Links
Guide pages should link to:
- 5-10 breed pages,
- comparison page,
- quiz,
- methodology.

### Comparison Internal Links
Comparison pages should link back to:
- each compared breed,
- related comparison pages,
- guide pages.

## 10. Korean SEO Notes

Korean pages should include natural phrases users actually search.

Examples:
- `메인쿤 성격`
- `랙돌 단점`
- `러시안블루 털빠짐`
- `브리티시 숏헤어 분양 전 확인`
- `초보 집사 고양이 추천`

Avoid overusing exact keywords unnaturally.

## 11. Image SEO

Each image needs:
- descriptive filename,
- alt text in page language,
- width and height,
- compression,
- lazy loading where appropriate,
- source status metadata.

Filename examples:
- `maine-coon-cat-full-body-ai-reviewed.webp`
- `ko-maine-coon-cat-portrait.webp`

Alt text examples:
- English: `Large Maine Coon cat with long brown tabby coat sitting indoors.`
- Korean: `실내에 앉아 있는 긴 갈색 태비 털의 메인쿤 고양이.`

## 12. SEO Quality Risks

### Risk: Thin Bilingual Duplication
Mitigation:
- Localize Korean content.
- Use Korean-specific FAQs.
- Avoid literal translation only.

### Risk: AI Image Trust Issue
Mitigation:
- Label generated images.
- Add image policy page.
- Do not claim generated images are real photos.

### Risk: Medical Misinformation
Mitigation:
- Use disclaimers.
- Cite reputable references.
- Avoid diagnosis or treatment advice.

### Risk: Over-Monetized Pages
Mitigation:
- Delay affiliate blocks until core content is strong.
- Keep product recommendations relevant and transparent.

## 13. Launch SEO Checklist

- All pages have unique titles.
- All pages have meta descriptions.
- All English/Korean equivalents have hreflang.
- Sitemap includes all indexable pages.
- Breed pages have FAQ sections.
- FAQ schema matches visible content.
- Images are compressed and have alt text.
- Internal links exist on all breed pages.
- No placeholder text remains.
- Methodology and image policy pages are published.
