# 체크리스트: 인기 상위 30종 마스터피스 리마스터링

## 📋 개요
사용자가 승인한 v1 설정(40 steps, CFG 1.0)을 기반으로, 공식 고양이 협회(TICA/CFA)의 표준 이미지를 참고하여 상위 30종의 에셋 품질을 극대화함.

## 🏗️ 작업 단계

### Phase 1: 인프라 및 데이터 수집
- [ ] `src/data/official_refs/` 디렉토리 생성 및 관리 구조 수립
- [ ] 상위 30종 품종 리스트 및 TICA/CFA 표준 외형 데이터 확정
- [ ] 공식 참조 이미지(Standard Photos) 웹 수집 및 로컬 백업

### Phase 2: 프롬프트 엔지니어링 (Hybrid Analysis)
- [ ] 30종 전체에 대한 하이브리드 디테일 매핑 (`BREED_DETAILS` 업데이트)
- [ ] 각 품종별 고유 유전적 특징(귀, 눈, 체형)을 영문 프롬프트로 정밀 기술

### Phase 3: 에셋 생성 (RTX 5080 최적화)
- [ ] `generate_masterpiece.py`를 상위 30종 순차 처리 모드로 전환
- [ ] v1 표준 적용: 40 Steps, CFG 1.0, 1024x1280 (Portrait)
- [ ] 생성 완료 후 `public/images/Nyan/` 폴더 내 결과물 검수

### Phase 4: 비교 및 검증 (User Review)
- [ ] 공식 참조 이미지와 생성 결과물을 매칭하는 비교 데이터셋 구축
- [ ] 사용자의 피드백에 따른 미세 조정 및 재생성 프로세스 가동

## 🚨 주의 사항
- 공식 이미지는 오직 '참조 및 비교용'으로만 사용하며, 생성 결과물은 독자적인 AI 아트워크여야 함.
- 파일 삭제 금지 철칙(`No Removal`)을 준수하며 모든 작업은 덮어쓰기 또는 신규 생성으로 진행.
