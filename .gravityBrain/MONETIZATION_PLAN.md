# Monetization Plan

## 1. Monetization Philosophy

The site should earn revenue without damaging trust. Cat breed research is a sensitive decision area because users may make long-term pet ownership decisions. Monetization must support informed ownership, not impulse buying.

Core principle:
- Help first, monetize second.

## 2. Revenue Phases

### Phase 1: Trust and Traffic
Goal:
- Build SEO authority and user trust.

Revenue:
- None or minimal.

Focus:
- Complete breed pages.
- Build comparison and quiz.
- Collect feedback.
- Add newsletter signup.

### Phase 2: Affiliate Recommendations
Goal:
- Add helpful product recommendations tied to actual care needs.

Categories:
- Cat food.
- Litter.
- Litter boxes.
- Brushes and grooming tools.
- Nail clippers.
- Cat carriers.
- Scratching posts.
- Cat trees.
- Toys and enrichment.
- Water fountains.
- Beds.
- Cleaning supplies.
- Pet insurance where regionally appropriate.

Placement rules:
- Do not place affiliate products above core educational content.
- Clearly disclose affiliate relationships.
- Match products to breed needs.
- Avoid recommending unnecessary products.

### Phase 3: Lead Magnets and Digital Products
Possible products:
- Adoption readiness checklist.
- First-time cat owner starter guide.
- Breed comparison worksheet.
- Grooming schedule template.
- New cat home preparation checklist.

### Phase 4: Sponsorships
Possible sponsors:
- Ethical cat product brands.
- Litter brands.
- Grooming brands.
- Pet insurance.
- Vet telehealth services.

Rules:
- Sponsorship must be disclosed.
- Sponsored content must be separated from editorial ratings.

### Phase 5: Directory or Marketplace
Only consider later.

Possible directories:
- Veterinarians.
- Groomers.
- Shelters.
- Adoption organizations.
- Ethical breeders.

High risk:
- Breeder directory can create ethical and legal concerns.
- Needs strict policy and review.

## 3. Affiliate Content Strategy

### Breed-Specific Starter Kit
Each breed page may include a small care kit block.

Example:
- Persian: slicker brush, comb, eye wipes, grooming table mat.
- Maine Coon: large cat tree, large carrier, heavy-duty brush.
- Bengal: puzzle toys, climbing shelves, interactive wand toys.
- Sphynx: gentle wipes, warm bed, skin-safe care products.

### Guide-Based Monetization
Better monetization pages:
- Best brushes for long-haired cats.
- Best cat trees for large cats.
- Best toys for active cats.
- Best litter boxes for big cats.
- Starter kit for first-time cat owners.

## 4. Disclosure Requirements

Required pages/sections:
- Affiliate disclosure.
- Editorial policy.
- Sponsorship policy if applicable.

Suggested disclosure text:
`Some links may be affiliate links. If you buy through them, we may earn a commission at no extra cost to you. We only recommend products that are relevant to the care topic discussed.`

Korean:
`일부 링크는 제휴 링크일 수 있습니다. 해당 링크를 통해 구매하면 추가 비용 없이 운영자가 일정 수수료를 받을 수 있습니다. 콘텐츠와 관련성이 있는 제품만 신중히 소개합니다.`

## 5. Monetization Do-Not-Do List

Avoid:
- Promoting impulse pet purchases.
- Ranking breeders without verification.
- Claiming products cure medical conditions.
- Overloading breed pages with ads.
- Hiding affiliate relationships.
- Recommending products unrelated to breed needs.

## 6. Conversion Paths

### Search Visitor Path
Breed page -> care section -> relevant starter kit -> affiliate click.

### Comparison User Path
Compare breeds -> understand care needs -> breed-specific guide -> product recommendation.

### Quiz User Path
Quiz result -> recommended breeds -> new cat checklist -> email signup.

## 7. Metrics

### Pre-Monetization
- Organic sessions.
- Email signup rate.
- Quiz completion rate.
- Comparison page usage.

### Monetization
- Affiliate CTR.
- Revenue per 1,000 sessions.
- Product block engagement.
- Guide page conversion rate.

### Trust Metrics
- Bounce rate.
- Return visits.
- Feedback quality.
- Low complaint rate about misleading content.

## 8. Best First Monetization Bets

Recommended order:
1. Grooming tools for long-haired breeds.
2. Cat trees and enrichment toys for active/large breeds.
3. Beginner starter kit checklist.
4. Litter and litter box guide.
5. Carriers and travel safety.

Reason:
- These are practical and directly connected to care needs.
